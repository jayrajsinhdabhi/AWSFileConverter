# AWS File Converter

Serverless image-to-PDF converter on AWS.

## What this app does

1. User uploads an image from the static web UI.
2. UI sends multipart form data to API Gateway at POST /convert.
3. Lambda converts the image to PDF, stores original and converted objects in S3.
4. Lambda returns a pre-signed download URL.
5. UI shows the download link.

## Supported conversion

| Source | Target |
|---|---|
| .jpg, .jpeg, .png, .webp, .gif, .tiff, .avif | .pdf |

## Required vs optional files

Required for deployment:

- template.yaml
- backend/src/handler.js
- backend/package.json
- ui/index.html
- ui/styles.css
- ui/app.js

Generated or optional:

- ui/config.js: generated after deploy (contains API URL)
- backend/coverage/: generated by tests
- lambda/: legacy prototype code, not used by template.yaml deployment path
- referencelambdafunc/: reference Lambda implementation folder you added for conversion logic examples
- amplify.yml: optional if you use Amplify hosting workflows

## Prerequisites

- AWS CLI configured with credentials/account
- AWS SAM CLI
- Node.js 20 or newer
- Docker (recommended for sam build --use-container)

## Deploy (PowerShell)

```powershell
# Install backend dependencies
Push-Location backend
npm install
Pop-Location

# Build Lambda artifact
sam build --use-container

# First deploy (guided)
sam deploy --guided

# Read stack outputs
$stackName = "aws-file-converter"
$apiUrl = aws cloudformation describe-stacks `
  --stack-name $stackName `
  --query "Stacks[0].Outputs[?OutputKey=='ApiUrl'].OutputValue" `
  --output text

$websiteBucket = aws cloudformation describe-stacks `
  --stack-name $stackName `
  --query "Stacks[0].Outputs[?OutputKey=='WebsiteBucketName'].OutputValue" `
  --output text

# Generate frontend config
"window.__CONFIG__ = { apiUrl: '$apiUrl' };" | Out-File -Encoding utf8 ui/config.js

# Upload frontend
aws s3 sync ui/ "s3://$websiteBucket/" --delete

# Get CloudFront URL
aws cloudformation describe-stacks `
  --stack-name $stackName `
  --query "Stacks[0].Outputs[?OutputKey=='CloudFrontUrl'].OutputValue" `
  --output text
```
